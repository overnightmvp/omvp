# Ramin Personal Training Template

> Enterprise-grade Astro template for personal trainers and service-based businesses.

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Fill in your details
#    Open template.config.ts and replace all {{PLACEHOLDERS}}

# 3. Add your content (see docs/TEMPLATE-GUIDE.md)
#    Edit files in src/content/

# 4. Start dev server
npm run dev

# 5. Build for production
npm run build
```

## Requirements

- Node.js 18+
- npm 9+

## What's Included

- **Astro 4** static site generation
- **Type-safe** content collections (Zod schemas)
- **Mobile-first** design with 44px+ touch targets
- **PWA** support (offline, installable)
- **WhatsApp** floating widget
- **6 CTA variants**, 4 sizes, full states
- **Blog** system with RSS feed
- **Dark/Light/High-contrast** themes
- **SEO** — schema markup, Open Graph, sitemap
- **13 documentation files**

## Folder Structure

```
src/
  content/        ← Your text content (Markdown / JSON)
  components/     ← Reusable UI components
  layouts/        ← Page wrappers
  pages/          ← Routes
  styles/         ← Design system CSS
template.config.ts  ← 🎯 Main config
docs/             ← All documentation
```
